MindLense
============

Plateforme de journaling personnel multimodal avec IA (Django).

Demarrage
---------

```bash
python -m venv .venv
./.venv/Scripts/Activate.ps1
pip install django
python manage.py migrate
python manage.py runserver 127.0.0.1:8010
```
